汪晓钰个人网站 - 快速开始指南

💡 重要提示： 这是为您生成的个人作品集网站，包含首页、作品集和简历三个主要页面。

🎯 第一步：查看网站

1 本地查看

双击 index.html 文件，在浏览器中打开网站。

2 访问不同页面

打开首页 打开作品集 打开简历

📁 网站文件结构

personal_website/
├── index.html # 网站首页
├── portfolio.html # 作品集页面
├── resume.html # 简历页面
├── css/
│ └── style.css # 样式文件
├── js/
│ └── main.js # 交互脚本
├── images/ # 图片目录（存放您的照片和作品）
├── portfolio/ # 作品集图片目录
├── README.md # 详细说明文档
├── DEPLOYMENT.md # 部署指南
└── 快速开始.html # 本文件

🛠️ 第二步：自定义内容

1 更新个人信息

使用文本编辑器（如VSCode、记事本等）打开HTML文件，修改：

  • 姓名、联系方式
  • 教育背景
  • 工作经历
  • 项目经验
  • 技能列表
2 添加您的作品

将您的作品图片放入 images/ 或 portfolio/ 目录，然后在HTML中更新图片路径。

3 更换照片

将您的证件照放入 images/ 目录，然后在HTML中替换照片路径。

🚀 第三步：部署到互联网

1 选择部署平台
  • GitHub Pages：免费，适合技术背景用户
  • Netlify：免费，简单拖拽即可部署
  • Vercel：免费，快速部署
2 详细部署步骤

请参考 DEPLOYMENT.md 文件中的详细指南。

📱 网站功能特点

  • 响应式设计：适配手机、平板、电脑所有设备
  • 现代界面：适合影视艺术专业的视觉风格
  • 完整功能：作品展示、在线简历、联系表单
  • 无需服务器：纯静态文件，可部署到任何平台
  • 易于修改：HTML/CSS/JS代码清晰，易于自定义

📞 技术支持

如需帮助：

  1. 查看 README.md 文件中的详细说明
  2. 检查浏览器控制台是否有错误信息
  3. 确保所有文件路径正确
  4. 清除浏览器缓存后重试
